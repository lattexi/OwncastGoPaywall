OBS Setup Instructions

  1. Start the container using the button above
  2. In OBS, go to Settings → Stream
  3. Set Service to "Custom"
  4. Paste the RTMP URL in the "Server" field
  5. Paste the Stream Key in the "Stream Key" field
  6. Click "Start Streaming" in OBS
  7. Set the stream status to "Live" below
